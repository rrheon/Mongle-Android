# UC08. 검색 (Search)

## 화면 설명
답변 및 질문 전문 검색 화면 (탭 3)

## 사전 조건
- 로그인 + 그룹 선택 완료

## UI 요소
- 검색 바: 돋보기 아이콘, "답변이나 질문 검색" 플레이스홀더, X 버튼 (텍스트 입력 시)
- 상태별 표시:
  - 기본: 몽글 로고 + "가족의 소중한 기록을 검색해보세요"
  - 1글자: "2글자 이상 입력해 주세요"
  - 결과 없음: 돋보기 + "[검색어]에 맞는 기록이 없어요"
  - 결과: 검색 건수 + 날짜별 그룹핑된 카드 (매칭 텍스트 초록 하이라이트)

---

## UseCase

### UC08-1. 검색 실행
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 검색어 2글자 이상 입력 |
| 정상 흐름 | 텍스트 입력 → 디바운스 (400ms) → 검색 API 호출 → 결과 표시 (검색어 하이라이트) |
| 결과 없음 | 매칭 기록 없음 → 빈 상태 메시지 |
| 결과 | 날짜별 그룹핑된 검색 결과 카드 표시 |

### UC08-2. 검색어 삭제
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | X 버튼 터치 |
| 정상 흐름 | 터치 → 검색어 초기화 → 기본 상태로 복귀 |
| 결과 | 기본 안내 화면 |

### UC08-3. 검색 결과 스크롤
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 결과 리스트 스크롤 |
| 정상 흐름 | 스크롤 → 결과 카드 탐색 (iOS: 11개 결과마다 광고 배너 삽입) |
| 결과 | 검색 결과 전체 탐색 |
