# UC07. 히스토리 (History)

## 화면 설명
캘린더 기반 과거 질문/답변 조회 화면 (탭 2)

## 사전 조건
- 로그인 + 그룹 선택 완료

## UI 요소
- 헤더: "기록" + 월 네비게이션 (좌/우 화살표, "YYYY년 M월")
- 캘린더 그리드: 요일 헤더 (일~토), 날짜 셀 (기록 있는 날 초록 점, 오늘 초록 원, 선택일 연초록)
- "최근 14일 기분" 섹션: 5개 기분 캐릭터 + 빈도 뱃지
- 선택된 날짜 상세: 질문 카드 + 멤버 답변 카드들
- 빈 상태: "이 날의 기록이 없어요" 또는 "오전 11시에 새로운 질문이 도착해요"

---

## UseCase

### UC07-1. 월 탐색
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 좌/우 화살표 버튼 터치 |
| 정상 흐름 | 화살표 터치 → 이전/다음 월로 전환 → 해당 월 데이터 로드 → 캘린더 갱신 |
| 결과 | 해당 월의 캘린더 및 기록 표시 |

### UC07-2. 날짜 선택
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 캘린더 날짜 셀 터치 |
| 정상 흐름 | 날짜 터치 → 해당 날짜의 질문 + 답변 카드 표시 |
| 빈 데이터 | 기록이 없는 날 → "이 날의 기록이 없어요" 표시 |
| 결과 | 선택된 날짜의 질문/답변 상세 표시 |

### UC07-3. 과거 질문 상세 보기
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 날짜 상세의 질문 카드 터치 |
| 정상 흐름 | 질문 카드 터치 → **질문 상세 화면으로 이동** |
| 결과 | 해당 질문의 상세 화면 진입 |

### UC07-4. 기분 통계 확인
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 화면 진입 시 자동 표시 |
| 정상 흐름 | 최근 14일간의 기분 데이터 집계 → 5개 캐릭터별 빈도 뱃지 표시 |
| 결과 | 기분 트렌드 시각화 |
