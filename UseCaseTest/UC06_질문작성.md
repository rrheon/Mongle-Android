# UC06. 질문 작성 (Write Question)

## 화면 설명
사용자가 직접 오늘의 질문을 작성하는 화면 (하트 3개 소모)

## 사전 조건
- 로그인 + 그룹 선택 완료
- 하트 3개 이상 보유 (홈에서 확인 팝업 통과)

## UI 요소
- 상단 바: 뒤로가기 + "질문 작성하기" 타이틀
- 설명: "나만의 질문을 작성해요" + "작성한 질문은 오늘의 질문으로 등록돼요."
- 텍스트 에디터 (플레이스홀더: 예시 질문, 최대 8줄)
- 글자 수 표시 ("N 자")
- CTA 버튼: "질문 등록하기" / "등록 중..." (빈 텍스트 또는 제출 중 비활성화)

---

## UseCase

### UC06-1. 질문 등록
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 질문 텍스트 입력 후 "질문 등록하기" 터치 |
| 정상 흐름 | 텍스트 입력 → "질문 등록하기" 터치 → API 호출 → 질문 등록 → 화면 닫힘 → 홈 복귀 + 성공 토스트 + 오늘의 질문 교체 + 하트 -3 |
| 검증 실패 | 텍스트 미입력 → 버튼 비활성화 (터치 불가) |
| 실패 흐름 | API 실패 → 에러 토스트 (3초 후 자동 사라짐) |
| 결과 (성공) | 나만의 질문 등록 + 하트 -3 |

### UC06-2. 작성 취소
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 뒤로가기 버튼 터치 |
| 정상 흐름 | 터치 → 화면 닫힘 → **홈 화면 복귀** |
| 결과 | 작성 취소, 하트 소모 없음 |
