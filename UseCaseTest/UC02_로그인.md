# UC02. 로그인 (Login)

## 화면 설명
소셜 로그인 및 둘러보기(게스트) 진입 화면

## 사전 조건
- 온보딩 완료 또는 건너뜀
- 인증된 세션 없음

## UI 요소
- 앱 로고 + "몽글" 타이틀
- 태그라인: "오늘의 마음은 어떤 색인가요?"
- 소셜 로그인 버튼: 카카오, 구글, 애플
- "둘러보기" 텍스트 버튼
- 로딩 오버레이 (로그인 진행 시 반투명 배경 + 스피너)

---

## UseCase

### UC02-1. 카카오 로그인
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 카카오 로그인 버튼 터치 |
| 정상 흐름 | 버튼 터치 → 카카오 SDK 인증 → 서버 로그인 API 호출 → 토큰 저장 → **그룹 선택 화면으로 이동** |
| 실패 흐름 1 | 카카오 인증 취소 → 무시 (화면 유지) |
| 실패 흐름 2 | 카카오 인증 실패 → 에러 토스트 표시 |
| 실패 흐름 3 | 서버 로그인 실패 → 에러 토스트 표시 |
| 결과 (성공) | 그룹 선택 화면 진입 |
| 결과 (실패) | 로그인 화면 유지 + 에러 메시지 |

| 항목 | 내용 |

| 실패 흐름 1 | 구글 인증 취소 → 무시 (화면 유지) |
| 실패 흐름 2 | 구글 인증 실패 → 에러 토스트 표시 |
| 실패 흐름 3 | 서버 로그인 실패 → 에러 토스트 표시 |
| 결과 (성공) | 그룹 선택 화면 진입 |
| 결과 (실패) | 로그인 화면 유지 + 에러 메시지 |

### UC02-3. 애플 로그인
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 애플 로그인 버튼 터치 |
| 정상 흐름 (iOS) | 버튼 터치 → Apple Sign-In → 서버 로그인 API 호출 → 토큰 저장 → **그룹 선택 화면으로 이동** |
| 정상 흐름 (Android) | 버튼 터치 → "Apple 로그인은 준비 중입니다." 에러 표시 |
| 실패 흐름 | 애플 인증 취소 → 무시 (화면 유지) |
| 결과 | iOS: 그룹 선택 화면 진입 / Android: 미구현 안내 |

### UC02-4. 둘러보기 (게스트)
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | "둘러보기" 텍스트 터치 |
| 정상 흐름 | 터치 → 게스트 모드 진입 → **메인 탭 화면으로 이동** (제한된 기능) |
| 제한 사항 | 답변, 넛지, 그룹 관리 등 주요 기능에서 "로그인이 필요해요" 팝업 표시 |
| 결과 | 읽기 전용 메인 화면 진입 |
