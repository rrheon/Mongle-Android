# UC11. 프로필/설정 (Profile / Settings / MY)

## 화면 설명
사용자 프로필, 앱 설정, 그룹 관리, 계정 관리를 포함하는 MY 탭 (탭 4)

## 사전 조건
- 로그인 완료

## UI 요소
### MY 메인
- 헤더: "MY"
- 프로필 카드: 기분 기반 몽글 캐릭터 + 사용자 이름 + 현재 기분
- 광고 배너
- 프로필 섹션: "프로필 수정"
- 앱 설정 섹션: "알림 설정", "그룹 관리", "계정 관리"
- 앱 버전: "몽글 v1.0.0"

### 프로필 수정
- 기분 캐릭터 아바타 (80dp)
- 안내: "답변 수정 시 색상을 변경할 수 있어요"
- 이름 텍스트 필드 (7일 쿨다운: "N일 후 변경 가능")
- "저장" 버튼

### 알림 설정
- 토글: 답변 알림, 재촉 알림, 시스템 알림
- 방해 금지 시간 (현재 비활성화)

### 그룹 관리
- 초대 코드 표시 + 복사/공유 버튼
- 멤버 리스트 (방장: 멤버 내보내기 가능)
- "그룹 나가기" 버튼
- 방장 나가기: 방장 위임 바텀시트 → 새 방장 선택 → 확인

### 계정 관리
- "로그아웃" 버튼
- "회원 탈퇴" 버튼

## UseCase

### UC11-1. 프로필 수정
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | "프로필 수정" → 이름 변경 → "저장" 터치 |
| 정상 흐름 | 이름 입력 → 저장 → API 호출 → 이름 변경 완료 |
| 실패 흐름 1 | 7일 쿨다운 미경과 → 입력 필드 비활성화 + "N일 후 변경 가능" |
| 실패 흐름 2 | API 실패 → 에러 토스트 |
| 결과 (성공) | 프로필 이름 변경 |

### UC11-2. 알림 설정 변경
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 알림 토글 스위치 조작 |
| 정상 흐름 | 토글 변경 → API 호출 → 설정 저장 |
| 결과 | 알림 설정 변경 완료 |

### UC11-3. 초대 코드 복사
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 그룹 관리 → 초대 코드 "복사" 버튼 터치 |
| 정상 흐름 | 터치 → 클립보드 복사 → "초대 코드가 복사되었어요" 토스트 |
| 결과 | 클립보드에 초대 코드 복사 |

### UC11-4. 초대 코드 공유
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 그룹 관리 → "공유" 버튼 터치 |
| 정상 흐름 | 터치 → 시스템 공유 시트 표시 |
| 결과 | 초대 코드 공유 |

### UC11-5. 멤버 내보내기 (방장)
| 항목 | 내용 |
|------|------|
| 액터 | 방장 |
| 트리거 | 멤버 리스트에서 내보내기 터치 |
| 정상 흐름 | 내보내기 터치 → 확인 팝업 → 확인 → API 호출 → 멤버 제거 |
| 결과 | 멤버 그룹에서 제거 |

### UC11-6. 그룹 나가기
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | "그룹 나가기" 버튼 터치 |
| 정상 흐름 (일반 멤버) | 터치 → 확인 팝업 → 확인 → API 호출 → **그룹 선택 화면으로 이동** |
| 정상 흐름 (방장, 다른 멤버 존재) | 터치 → 방장 위임 바텀시트 → 새 방장 선택 → 확인 → API 호출 → **그룹 선택 화면으로 이동** |
| 정상 흐름 (방장, 혼자) | 터치 → 그룹 삭제 확인 → 확인 → API 호출 → **그룹 선택 화면으로 이동** |
| 결과 | 그룹 탈퇴/삭제 + 그룹 선택 화면 이동 |

### UC11-7. 로그아웃
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 계정 관리 → "로그아웃" 터치 |
| 정상 흐름 | 터치 → 확인 팝업 ("정말 로그아웃 하시겠어요?") → 확인 → 세션 삭제 → **로그인 화면으로 이동** |
| 취소 | 확인 팝업에서 취소 → 화면 유지 |
| 결과 (성공) | 로그아웃 + 로그인 화면 이동 |

### UC11-8. 회원 탈퇴
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 계정 관리 → "회원 탈퇴" 터치 |
| 정상 흐름 | 터치 → 경고 팝업 ("모든 데이터가 삭제됩니다. 되돌릴 수 없습니다.") → 확인 → 소셜 연동 해제 → 계정 삭제 API → **로그인 화면으로 이동** |
| 취소 | 확인 팝업에서 취소 → 화면 유지 |
| 실패 흐름 | API 실패 → 에러 토스트 |
| 결과 (성공) | 계정 영구 삭제 + 로그인 화면 이동 |
