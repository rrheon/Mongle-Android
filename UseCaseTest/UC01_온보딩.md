# UC01. 온보딩 (Onboarding)

## 화면 설명
앱 최초 실행 시 표시되는 3페이지 캐러셀 튜토리얼

## 사전 조건
- 앱 최초 설치 후 첫 실행
- 또는 "다시 보지않기"를 선택하지 않은 상태

## UI 요소
- 3페이지 수평 스와이프 캐러셀
  - 페이지 1: 로고 + "몽글에 오신 걸 환영해요" + 가족/친구 소통 설명
  - 페이지 2: 그룹 카드 일러스트 + "나만의 공간을 만들어보세요"
  - 페이지 3: 질문 카드 + 5개 캐릭터 + "매일 함께 마음을 나눠요"
- 페이지 인디케이터 (3개 점, 선택된 점은 넓게)
- 메인 버튼: "다음" / "시작하기" / "몽글 시작하기" (페이지별 변경)
- "다시 보지않기" 텍스트 버튼

---

## UseCase

### UC01-1. 페이지 스와이프
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 화면 좌우 스와이프 |
| 정상 흐름 | 스와이프 → 다음/이전 페이지로 전환 → 인디케이터 업데이트 |
| 결과 | 해당 페이지 콘텐츠 표시 |

### UC01-2. 다음 버튼 터치
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 메인 버튼 터치 |
| 정상 흐름 (1~2페이지) | 버튼 터치 → 다음 페이지로 이동 |
| 정상 흐름 (3페이지) | "몽글 시작하기" 터치 → 온보딩 완료 → **로그인 화면으로 이동** |
| 결과 | 페이지 전환 또는 로그인 화면 진입 |

### UC01-3. 다시 보지않기
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | "다시 보지않기" 텍스트 터치 |
| 정상 흐름 | 터치 → 설정 저장 (UserDefaults/SharedPreferences) → **로그인 화면으로 이동** |
| 결과 | 다음 앱 실행 시 온보딩 건너뜀 |
