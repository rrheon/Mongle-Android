# UC04. 홈 (Home)

## 화면 설명
오늘의 질문, 가족 몽글 캐릭터 씬, 그룹 전환 등 앱의 메인 화면

## 사전 조건
- 로그인 + 그룹 선택 완료 상태

## UI 요소
- 상단 바: 그룹 이름 드롭다운, 하트 수 뱃지, 알림 벨 (읽지 않은 알림 시 빨간 점)
- 오늘의 질문 카드 (답변 완료 시 체크마크)
- 몽글 씬: 가족 멤버 캐릭터 애니메이션 (답변/미답변 상태별 표시)
- 질문 미도착 시: "오전 11시에 새로운 질문이 도착해요" 플레이스홀더

---

## UseCase

### UC04-1. 오늘의 질문 카드 터치
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 질문 카드 터치 |
| 정상 흐름 | 터치 → **질문 시트 바텀시트 표시** (답변하기, 나만의 질문 작성, 질문 넘기기 옵션) |
| 결과 | 바텀시트 표시 |

### UC04-2. 답변하기
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 질문 시트에서 "답변하기" 터치 |
| 정상 흐름 | 터치 → **질문 상세 화면으로 이동** |
| 결과 | 질문 상세 화면 진입 |

### UC04-3. 나만의 질문 작성
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 질문 시트에서 "나만의 질문 작성하기" 터치 |
| 정상 흐름 | 터치 → 하트 3개 소모 확인 팝업 → 확인 → **질문 작성 화면으로 이동** |
| 실패 흐름 | 하트 부족 → "하트가 부족해요" 팝업 |
| 결과 (성공) | 질문 작성 화면 진입 |

### UC04-4. 질문 넘기기
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 질문 시트에서 "질문 넘기기" 터치 |
| 정상 흐름 | 터치 → 하트 3개 소모 확인 팝업 → 확인 → API 호출 → 질문 넘김 완료 |
| 실패 흐름 1 | 하트 부족 → "하트가 부족해요" 팝업 |
| 실패 흐름 2 | API 실패 → 에러 토스트 |
| 결과 (성공) | 오늘 질문 넘김 상태, 하트 -3 |

### UC04-5. 그룹 전환
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 상단 그룹 이름 드롭다운 터치 |
| 정상 흐름 | 드롭다운 열림 → 다른 그룹 선택 → 홈 데이터 새로고침 |
| 추가 옵션 | "그룹 관리" → **그룹 선택 화면으로 이동** |
| 결과 | 선택된 그룹의 홈 데이터 표시 |

### UC04-6. 하트 정보 확인
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 하트 뱃지 터치 |
| 정상 흐름 | 터치 → 팝오버/팝업 표시 (하트 사용처: 질문 다시받기 1, 나만의 질문 3, 재촉 1 / 충전: 매일 +1, 답변 완료 +1) |
| 결과 | 하트 경제 시스템 안내 |

### UC04-7. 알림 확인
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 알림 벨 아이콘 터치 |
| 정상 흐름 | 터치 → **알림 화면으로 이동** |
| 결과 | 알림 목록 화면 진입 |

### UC04-8. 답변한 멤버 캐릭터 터치
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 답변 완료한 다른 멤버의 몽글 캐릭터 터치 |
| 전제 조건 | 본인이 이미 답변 완료 |
| 정상 흐름 | 터치 → **피어 답변 바텀시트** 표시 (상대 멤버의 답변 내용) |
| 실패 흐름 | 본인 미답변 → "먼저 답변을 완료해 주세요" 팝업 |
| 결과 | 멤버 답변 확인 |

### UC04-9. 미답변 멤버 캐릭터 터치
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 미답변 멤버의 몽글 캐릭터 터치 |
| 정상 흐름 | 터치 → **넛지(재촉) 화면으로 이동** |
| 실패 흐름 | 본인 미답변 → "먼저 답변을 완료해 주세요" 팝업 |
| 결과 | 넛지 화면 진입 |

### UC04-10. 일일 하트 지급
| 항목 | 내용 |
|------|------|
| 액터 | 시스템 |
| 트리거 | 인증 후 첫 홈 화면 진입 (하루 1회) |
| 정상 흐름 | 홈 데이터 로드 시 일일 하트 클레임 → "하트 +1" 팝업 표시 |
| 결과 | 하트 +1 지급 |
