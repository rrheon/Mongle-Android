# UC10. 넛지/재촉 (Peer Nudge)

## 화면 설명
미답변 멤버에게 재촉 메시지를 보내는 화면 (하트 1개 소모)

## 사전 조건
- 로그인 + 그룹 선택 + 본인 답변 완료
- 대상 멤버가 미답변 상태

## UI 요소
- 그라데이션 배경 (따뜻한 피치 → 초록)
- 상단 바: 뒤로가기 + "재촉하기" + 하트 수 뱃지
- 대상 멤버 캐릭터 아바타
- "[이름]에게 재촉 메시지를 보낼까요?"
- 하트 소모 뱃지: "하트 1개 소모"
- 상태별 표시:
  - 하트 충분: "재촉하기" 초록 버튼
  - 하트 부족: "하트가 부족해요" + "광고 보고 재촉하기" 버튼
  - 전송 완료: "재촉 메시지를 보냈어요!" 성공 카드

---

## UseCase

### UC10-1. 재촉 메시지 보내기
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | "재촉하기" 버튼 터치 |
| 정상 흐름 | 터치 → API 호출 → 넛지 전송 → "재촉 메시지를 보냈어요!" 표시 + 하트 -1 |
| 실패 흐름 | API 실패 → 에러 토스트 |
| 결과 (성공) | 재촉 전송 + 하트 -1 + 버튼 비활성화 |

### UC10-2. 광고 시청 후 재촉
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | "광고 보고 재촉하기" 버튼 터치 (하트 부족 시) |
| 정상 흐름 | 터치 → 보상형 광고 시청 → 광고 완료 → 재촉 전송 (무료) |
| 실패 흐름 | 광고 로드 실패 → 에러 토스트 |
| 결과 (성공) | 재촉 전송 (하트 소모 없음) |

### UC10-3. 재촉 취소
| 항목 | 내용 |
|------|------|
| 액터 | 사용자 |
| 트리거 | 뒤로가기 버튼 터치 |
| 정상 흐름 | 터치 → **홈 화면 복귀** |
| 결과 | 재촉 미발송, 하트 소모 없음 |
